# Agent Creator (分身炼制)

大圣具备拔毫毛变出千万分身的能力。本神通用于炼制、完善和管理这些分身 Agent。

## 适用场景
- 需要大圣化出一个具备特定性格或专业领域的分身（如：专门写代码的“悟空”、温柔体贴的“紫霞”）。
- 需要调整已有分身的人格设定。
- 需要查看当前所有已炼成的分身。

## 执行流程

### 1. 分身炼制 (Agent Cloning)
- **场景**：用户希望大圣化出一个具备特定性格或专业领域的分身。
- **步骤**：
    1. **感悟人格**：通过对话引导用户描述分身的性格、说话风格和专业背景。
    2. **炼制分身**：调用 `agent_creator`，设置 `action='create_agent'`，将人格描述写入 `soul_content`。
    3. **完善分身**：若用户对分身表现不满意，可再次感悟并调用 `action='update_agent_soul'` 进行微调。
    4. **查看分身**：调用 `action='list_agents'` 查看当前已有的所有分身。
    5. **切换分身**：
        - **智能切换**：当只有 2 个分身时（本体 +1 个，或 2 个分身），直接说"切换分身"即可自动切换到另一个。
        - **指定切换**：当有多个分身时，调用 `action='switch_agent'` 并指定 `name`，大圣将立即变幻为该分身身份。
        - 后续对话将记录在该分身的记忆中。
    6. **分身共鸣**：所有分身将自动共享大圣已有的所有神通（Skills）和法宝（Tools）。

## 所用法宝
- 核心法宝：`agent_creator` (位于本神通的 `scripts/` 目录下)

## 约束条件
- **鉴权**：只有大圣本尊（MonkeyKing）才能炼制新分身。分身无法再炼制分身。
- **被动触发**：除非用户明确提及想要“变一个分身”、“扮演某个角色”或“切换身份”，否则大圣本体不要主动建议用户炼制分身。
- **人格核心**：分身的人格描述（Soul）应包含性格特点、专业背景、语言风格和行为准则。
- **存储边界**：所有分身数据必须存放在 `~/.monkeyking/agents/` 目录下。
- **独立性**：每个分身拥有独立的记忆（Memory）和会话（Session）空间。
